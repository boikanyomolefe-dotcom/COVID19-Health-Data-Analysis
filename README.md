# COVID-19 Health Data Analysis

## Project Overview
Analysis of a COVID-19 health survey dataset using Python.

## Tools Used
- Python 3.13
- pandas
- seaborn
- matplotlib

## What I did
- Loaded and cleaned real health survey data
- Performed statistical analysis by age group and gender
- Created visualizations including countplots, histograms, boxplots and violin plots
